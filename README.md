# 🧠 Clinical AI Assistant

AI-powered clinical reasoning assistant that transforms unstructured clinical input into structured analytical insights using Large Language Models (LLMs).

---

## 🚀 Project Overview

Clinical information is often messy, incomplete, and difficult to interpret efficiently. This project explores how AI systems can move beyond passive chatbot-style responses toward structured reasoning and intelligent information synthesis.

The system accepts free-text clinical-style input and generates organized analytical outputs including:
- symptom extraction
- possible condition inference
- reasoning pathways
- suggested actions
- confidence estimation

This project was built as an exploration of:
- agentic AI systems
- human–AI interaction
- structured reasoning workflows
- AI-assisted clinical decision support

## ▶️ Running the Project Locally

### Clone Repository

```bash
git clone https://github.com/Harshalspage/clinical-ai-assistant.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python -m streamlit run app.py
```
